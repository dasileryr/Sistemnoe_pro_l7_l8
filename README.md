# Практические задания по системному программированию (L7-L8)

## Описание

Этот проект содержит решения заданий по работе с потоками в C++.

## Структура проекта

- **task1.cpp** - Задание 1: Поток выводит числа от 0 до 50
- **task2.cpp** - Задание 2: Поток выводит числа в заданном пользователем диапазоне
- **task3.cpp** - Задание 3: Несколько потоков выводят числа в заданном диапазоне
- **task4.cpp** - Задание 4: Поиск максимума, минимума и среднего в массиве из 10000 элементов
- **task5.cpp** - Задание 5: Задание 4 + запись результатов в файл
- **task6.cpp** - Задание 6: Оконный интерфейс для заданий 1-3 (Qt)
- **task7.cpp** - Задание 7: Оконный интерфейс для заданий 4-5 (Qt)

## Компиляция

### Консольные приложения (задания 1-5)

```bash
make console
```

Или компиляция отдельных заданий:

```bash
g++ -std=c++11 -Wall -Wextra -pthread -o task1 task1.cpp
g++ -std=c++11 -Wall -Wextra -pthread -o task2 task2.cpp
g++ -std=c++11 -Wall -Wextra -pthread -o task3 task3.cpp
g++ -std=c++11 -Wall -Wextra -pthread -o task4 task4.cpp
g++ -std=c++11 -Wall -Wextra -pthread -o task5 task5.cpp
```

### Оконные приложения (задания 6-7)

Требуется установленный Qt5:

```bash
# Ubuntu/Debian
sudo apt-get install qt5-default libqt5widgets5-dev

# macOS (с Homebrew)
brew install qt5

# Затем компиляция
make gui
```

Или вручную:

```bash
g++ -std=c++11 -Wall -Wextra -pthread -fPIC $(pkg-config --cflags Qt5Widgets) -o task6 task6.cpp $(pkg-config --libs Qt5Widgets)
g++ -std=c++11 -Wall -Wextra -pthread -fPIC $(pkg-config --cflags Qt5Widgets) -o task7 task7.cpp $(pkg-config --libs Qt5Widgets)
```

## Описание заданий

### Задание 1
Создано консольное приложение, порождающее поток. Поток отображает в консоль числа от 0 до 50.

### Задание 2
Добавлена возможность передачи начала и конца диапазона чисел. Границы определяет пользователь через консольный ввод.

### Задание 3
Добавлена возможность определения пользователем количества потоков. Границы диапазона чисел также выбираются пользователем. Диапазон равномерно распределяется между потоками.

### Задание 4
Консольное приложение генерирует набор чисел, состоящий из 10000 элементов. С помощью механизма потоков находится максимум, минимум и среднее в этом наборе. Для каждой задачи выделен отдельный поток.

### Задание 5
К четвертому заданию добавлен поток, выводящий набор чисел и результаты вычислений в файл `results.txt`.

### Задание 6
Первое задание со всеми изменениями переведено в оконный интерфейс с использованием Qt. Пользователь может задать начало и конец диапазона, количество потоков, запустить/остановить выполнение и видеть результаты в реальном времени.

### Задание 7
Четвертое задание со всеми изменениями переведено в оконный интерфейс с использованием Qt. Пользователь может сгенерировать массив, запустить вычисления, увидеть результаты и сохранить их в файл.

## Требования

- C++11 или выше
- Компилятор с поддержкой std::thread (g++, clang++)
- Для заданий 6-7: Qt5 (Qt5Widgets)

## Очистка

```bash
make clean
```

Удаляет все скомпилированные файлы и временные файлы.
